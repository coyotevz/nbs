Consejo del Día

Nobix tiene un sistema de gestión de tareas pendientes. Puede ser útil para gestionar trabajos pendientes con los clientes o para el control de quehaceres internos de su empresa. Para más información vea el menú 'Tareas'.

Administre los clientes, proveedores y usuarios del sistema. Puede dar de alta nuevos clientes o proveedores si lo necesita.

Comprobantes

Administre diferentes tipos de comprobantes como facturas, tickets, recibos, remitos y presupuestos.

Ayuda

Encuentre toda la ayuda que necesita para operar el sistema de la manera más eficiente.
