Bienvenidos a la documentación de Nobix
=======================================

Nobix es un sistema para PyMEs que se dedican a la comercialización mayorista y
minorista de productos ayudando a realizar tareas administrativas, comerciales
y de gestión de un modo preciso y sencillo.
